# 🎓 ERP-Based College Management System

An AI-enabled ERP platform designed to simplify and automate college management processes such as student admissions, faculty management, hostel allocation, attendance tracking, academic records, and AI-powered student support.

> Developed as a Smart India Hackathon (SIH) project to provide a scalable and low-code ERP solution for educational institutions, especially rural colleges.

---

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Khushikri29/ERP_BASED-COLLEGE-MANAGEMENT-SYSTEM.git
```

Move into the project

```bash
cd ERP_BASED-COLLEGE-MANAGEMENT-SYSTEM
```

---

## Backend Setup

```bash
cd Backend
npm install
npm start
```

---

## Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```
